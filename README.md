简化项目中的序列化

- 开启组件方便传递参数
- 创建自动读写自本地的字段
- 创建自动读写Intent/Argument参数
- 创建意外销毁自动保存/恢复的字段
- 创建配置变更/屏幕旋转自动恢复的数据实例
- 快捷打开意图

<br>

在项目根目录的 build.gradle 添加仓库

```groovy
allprojects {
    repositories {
        // ...
        maven { url 'https://jitpack.io' }
    }
}
```

在 module 的 build.gradle 添加依赖

```groovy
implementation 'com.github.liangjingkanji:Serialize:1.0.4'
```
